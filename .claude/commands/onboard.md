# Onboard Command

Quickly onboard a new agent to the current project state.

## Instructions

1. Read CLAUDE.md for project configuration
2. Read PRD.md for full requirements
3. Read progress.md for current status
4. Check git log for recent commits
5. Scan the project structure
6. Identify the current module/phase
7. Report readiness to continue

## What to Report

- Current module in progress
- Tasks completed vs remaining
- Any blocking issues from progress.md
- Recent changes from git history

## Usage

```
/onboard
```

This will ground you in the project context so you can continue where the previous agent left off.
